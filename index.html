Clew

Precise, short distance navigation for the blind and visually impaired.

Clew uses ARKit and Google Cloud Anchors to create crowdsourced indoor routes, making spaces accessible to blind and visually impaired users. As a team member in the spring, I redesigned the app's front-end to improve usability and accessibility, focusing on an intuitive, low-vision-friendly interface and 100% screen reader accessibility. I also redesigned the codebase for better performance and modularity using the MVVM framework. As front-end development lead in the summer, I led the front-end integration of cloud anchors and developed new flows for recording and connecting anchors, choosing a route, and following the route to support the new feature. I also designed and developed a non-visual 'map' feature to accompany the step-by-step navigation. I mentored junior members on code maintainability, pair programming, and Git workflows, and together we implemented 10+ builds following continuous integration and feature-driven development practices.

Find and Replace Program

Text search and replace run on the command line.

A word search and replace program using the Knuth-Morris-Pratt (KMP) string search algorithm, which efficiently finds phrases in a large body of text by preprocessing the search term to skip unnecessary character comparisons. I utilized a test driven development approach to design a replace program that can handle complex queries, such as the replacement phrase containing the original phrase (e.g., replacing 'the' with 'there'). I designed the program to sequentially find and replace each instance of the search term without the need to store any locations or copies of the text, which optimizes the runtime and has low memory needs. My implementation includes commands for case-insensitive searches and whole word matching, allowing for precise and flexible text processing.

CYK Algorithm

Parsing context-free grammars.

An implementation of CYK parsing for context-free grammars, an algorithm that determines whether a given string is in the language of a grammar. This fascinating algorithm is foundational for natural language processing, as it enables computers to understand and process the structure of human languages by efficiently analyzing and interpreting grammatical constructs. My partner and I implemented the algorithm in Haskell, including a step-by-step command line print out used to demonstrate the inner workings to a class of 20 students.
